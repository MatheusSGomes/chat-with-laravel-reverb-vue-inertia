# Chat com Laravel e VueJS

## Comandos execução
```bash
php artisan serve
```

```bash
npm run dev
```

```bash
php artisan reverb:start --debug
```

## Instalação Laravel Echo no VueJS:
```bash
npm install --save-dev --force laravel-echo pusher-js @laravel/echo-vue && npm run build
```

## Para criar usuários:
```bash
php artisan tinker
```

```bash
User::factory(6)->create();
```
